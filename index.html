POST FLAGGER

Overview

  • Administrators can create flags like 'Favorites' or 'Seen' for logged in users
  • Allow customization of HTML code for each flag
  • Insert in themes and post by [shortcode]
  • This plugin creates 'Favorites' flag by default

Usage

Go to Settings / Post Flags for create, edit and delete flags

Insert flags in your code or post using the shortcode and the flag-slug: [flag slug="flag-slug"]

Theme Example

This example shows how to insert 'Favorites' default flag code in your theme files, where 'favorites' is the flag-slug. 
<?php do_shortcode('[flag slug="favorites"]'); ?>
